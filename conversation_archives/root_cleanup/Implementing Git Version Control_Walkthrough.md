## 今後の活用方法

今後、同様の「以前の状態に戻したい」という事態が発生した際、以下のコマンドで簡単に対処できます。

### 1. 変更を破棄して最後の大事な保存ポイントに戻る
誤ってレイアウトが崩れたり、実験が失敗した場合は以下のコマンドで最後にコミットした状態に即座に復旧できます。
```bash
git checkout .
```

### 2. 現在の状態を新しく保存する
大きな変更を加える前や、作業が一段落した際に実行します。
```bash
git add .
git commit -m "保存内容のメモ（例：記事リストのデザイン完成）"
```

### 3. 状態を確認する
現在、最後に保存したときから何が変更されているか確認します。
```bash
git status
```

### 4. 履歴を確認する
これまでに作成した保存ポイントの履歴を表示します。
```bash
git log --oneline
```

> [!TIP]
> 今後は、大きな実験的なデザイン変更を行う前に `git commit` を行っておくことで、いつでも「True North（真北/正しい指針）」の状態に立ち返ることが可能になります。
